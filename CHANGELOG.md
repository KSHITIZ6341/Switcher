# Changelog

All notable changes to Switcher will be documented in this file.

## [Unreleased]

## [1.1.0] - 2026-07-10

- Renamed the SwiftPM product, executable, app window, menu bar item, docs, and test imports to Switcher.
- Added resizable sidebar width with saved per-app width preferences.
- Added a pinned-app manager for focusing, reordering, unpinning, moving, and showing/hiding the active sidebar.
- Added search to the running-window picker.
- Updated the global hotkey so it opens pinning when idle and toggles the active sidebar when pinned.
- Added open-source project metadata, including license, contribution guide, security policy, code of conduct, and GitHub templates.

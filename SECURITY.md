# Security Policy

## Supported versions

Until versioned releases are published, only the current `main` branch is supported for security fixes.

## Reporting a vulnerability

Please do not report security vulnerabilities in public issues.

Use GitHub's private vulnerability reporting for this repository if it is available. If not, email the maintainer at Kshitizsuwal@gmail.com.

Include as much detail as you can:

- Affected macOS version.
- Steps to reproduce.
- Expected and actual impact.
- Whether Accessibility permissions or launch-at-login behavior are involved.
- Any proof-of-concept code, logs, screenshots, or screen recordings.

The maintainer will review the report, confirm the impact when possible, and coordinate a fix before public disclosure.
